Precalculus - Unit 1

Section 1.1 Investigation

Function Notation & Solving

Investigation: What is the difference between finding \(f(a)\) and solving \(f(x)=a\)?

Time: 15-20 minutes

Format: Partner investigation, representation comparison, and whole-class discussion

Learning Target: Evaluate functions and solve equations of the form \(f(x)=a\) using tables, algebraic, and graphical methods.
Investigation Directions

Setup

  • Grouping: Partners or groups of 3.
  • Materials: Student handout, pencil, and optional graphing calculator or Desmos.
  • Teacher display: Project one situation at a time during the share-out.
  • Timing: Assign each group 2 situations. Groups analyze their assigned pages, then use the summary page to compare all 8 situations during discussion.

Student Goal

Today you will compare contexts, tables, graphs, and equations. Your job is to discover what function notation is asking you to do.

\[\text{Evaluate: } f(a) \qquad \text{Solve: } f(x)=a\]

By the end, you should be able to explain why evaluating gives an output, while solving gives input value(s).

Norms for the Investigation

  • Use at least two representations before deciding on an answer.
  • Circle whether each question asks for an input or an output.
  • When solving \(f(x)=a\), check whether there is one solution, two solutions, or no solution.
  • Be ready to explain which representation was most efficient and why.

Class Notes / Final Rule
Teaching Resource 1: Teacher Script

Purpose

This investigation helps students discover the central structure of Section 1.1: evaluating a function means using an input to find an output, while solving \(f(x)=a\) means using an output to find input value(s). The task intentionally uses linear and quadratic situations so students see why some equations have one solution and others can have two.

Launch: 2 minutes

Say: “Today the main question is not just how to calculate. The main question is: what is the notation asking for? Are we finding an output, or are we finding input value(s)?”

Assign: Give each group two situations. Try to give each group one linear and one quadratic situation if possible.

Investigation: 10-12 minutes

  1. Circulate and ask, “Is the number inside parentheses an input or an output?”
  2. When students solve \(f(x)=a\), ask, “Where do you see that output in the table or graph?”
  3. For quadratics, press students to look for a second input with the same output.
  4. Ask groups to label each method as table, graph, equation, or context.
  5. Choose 2 linear and 2 quadratic groups to share during the discussion.

Synthesis: 5 minutes

Final summary: \(f(a)\) gives an output for a specified input. Solving \(f(x)=a\) gives the input value or values that produce a specified output. On a graph, evaluating starts at an x-value and reads the y-value. Solving starts at a y-value and finds the x-value(s).

Closure prompt: “How would you explain the difference between \(f(2)=8\) and \(f(x)=8\) to a student who was absent?”

Common Misconceptions to Watch For

  • Students may treat \(f(4)\) as multiplication.
  • Students may answer \(f(x)=a\) with a y-value instead of an x-value.
  • Students may stop after finding one quadratic solution when there are two.
  • Students may not connect table rows to ordered pairs on the graph.
  • Students may think context answers do not need units or interpretation.

Discussion Questions
  1. Difference: Evaluating uses a known input to find an output. Solving uses a known output to find input value(s).
  2. Representations: Tables and equations are often efficient for evaluating. Graphs and tables are often helpful for solving when the target output is visible.
  3. Linear vs. quadratic: A linear function with a nonzero slope usually reaches an output once. A quadratic can reach the same output on both sides of its vertex.
  4. Verify: Substitute the solution into the equation, check the table row, or see whether the graph has the target y-value at that x-value.
  5. Student statement: Both involve the output 10, but \(f(4)=10\) tells the output when input is 4, while \(f(x)=10\) asks which input or inputs produce output 10.
