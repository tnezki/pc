Unit 2.3 — Formative Spiral: Function Composition
5.

Decide whether each expression represents function composition or function arithmetic.

  1. \(f(g(x))\)
  2. \(f(x)+g(x)\)
  3. \((f\circ g)(x)\)
  4. \(f(x)g(x)\)

  2. \((g\circ f)(4)\)
