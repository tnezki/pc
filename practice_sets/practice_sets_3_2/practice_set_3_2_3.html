Unit 2.2 — Review Spiral: Function Arithmetic
13.

Let \(p(x)=x^2\) and \(q(x)=3x\).

  1. Find \((p+q)(4)\).
  2. Find \((p-q)(-2)\).
  3. Find \((pq)(1)\).
14.

Given:

\(f(x)=x^2-9\qquad g(x)=x-3\)

  1. Simplify \(\frac{f(x)}{g(x)}\).
  2. Explain why the domain restriction still exists after simplification.
15.

Let \(f(x)=\sqrt{x+1}\) and \(g(x)=6-x\).

  1. Find the domain of \(f+g\).
  2. Find the domain of \(fg\).
  3. Find the domain of \(\frac{f}{g}\).
  4. Explain why the quotient has one additional restriction.
16.

Design a pair of functions \(f\) and \(g\) where \(\frac{f}{g}\) simplifies algebraically but still has a hole in its graph.

  1. Write \(f(x)\) and \(g(x)\).
  2. Simplify \(\frac{f}{g}\).
  3. State the original domain restriction.
  4. Explain what the graph would show.
