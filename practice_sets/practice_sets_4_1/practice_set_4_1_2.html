Unit 2.1 — Formative Spiral: Odd and Even Functions
5.

7.

  1. A graph symmetric about the \(y\)-axis.
  2. A graph symmetric about the origin.
  3. A graph with no symmetry about the \(y\)-axis or origin.
8.

Fill in each blank.

  1. If \(f(-x)=f(x)\), the function is ____________________.
  2. If \(f(-x)=-f(x)\), the function is ____________________.

  2. Does adding a constant change whether the function is even?
