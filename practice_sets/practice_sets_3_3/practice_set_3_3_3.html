Unit 3.3 — Solve Cube Root Equations

Use the table to estimate the solution to \(\sqrt[3]{x+1}=x-1\).

\(x\)\(0\)\(1\)\(2\)\(3\)\(4\)
\(\sqrt[3]{x+1}\)\(1\)\(\sqrt[3]{2}\)\(\sqrt[3]{3}\)\(\sqrt[3]{4}\)\(\sqrt[3]{5}\)

  1. Explain why every horizontal line intersects exactly once.

Unit 2.3 Review Spiral — Function Composition
13.

Let \(f(x)=x^2\) and \(g(x)=x-4\). Evaluate:

  1. \(f(g(3))\)
  2. \(g(f(3))\)
14.

Let \(f(x)=\frac{1}{x}\) and \(g(x)=x-2\).

  1. Find \(f(g(x))\).
  2. State the domain restriction.
  3. Explain what input causes the problem.
15.

A student claims, “Function composition is commutative because \(f(g(x))\) and \(g(f(x))\) use the same two functions.”

  1. Test the claim using an example.
  2. Decide whether the claim is true.
  3. Write a complete explanation.
16.

Design a real-world situation involving two steps where the order matters.

  1. Define both functions.
  2. Write both compositions.
  3. Explain why one order matches the situation and the other does not.
