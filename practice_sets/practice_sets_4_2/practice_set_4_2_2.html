Unit 2.2 — Formative Spiral: Function Arithmetic
5.

Let \(p(x)=x^2\) and \(q(x)=3x\).

  1. Find \((p+q)(4)\).
  2. Find \((p-q)(-2)\).
  3. Find \((pq)(1)\).
6.

Fill in the blanks.

  1. \((f+g)(x)=\) ____________________
  2. \((f-g)(x)=\) ____________________
  3. \((fg)(x)=\) ____________________
  4. \(\left(\frac{f}{g}\right)(x)=\) ____________________, where \(g(x)\neq\) ________.
7.

Let \(f(x)=\sqrt{x}\) and \(g(x)=x+1\).

  1. Find \((f+g)(9)\).
  2. Find \((fg)(4)\).
  3. Explain why \(x=-1\) is not useful for evaluating \(f(x)\).
8.

Let \(f(x)=x^2-9\) and \(g(x)=x-3\).

  1. Find \(\left(\frac{f}{g}\right)(x)\).
  2. Simplify the expression.
  3. State the excluded value before simplification.
9.

Let \(f(x)=\sqrt{x+2}\) and \(g(x)=x-1\).

  1. Find \((f+g)(x)\).
  2. Find \((fg)(x)\).
  3. State the domain of each combined function.
10.

A student simplified \(\frac{x^2-9}{x-3}\) to \(x+3\) and said the domain is all real numbers.

  1. What did the student do correctly?
  2. What domain detail did the student miss?
  3. Write the correct simplified expression and domain restriction.
11.

Let \(f(x)=x^2-4\) and \(g(x)=x-2\).

  1. Find \(\left(\frac{f}{g}\right)(x)\) and simplify.
  2. Explain why the simplified expression does not have the same domain as \(x+2\).
  3. Describe what would happen on a graph at the excluded input.
12.

Design a pair of functions \(f\) and \(g\) where \(\frac{f}{g}\) simplifies algebraically but still has a hole in its graph.

  1. Write \(f(x)\) and \(g(x)\).
  2. Simplify \(\frac{f}{g}\).
  3. State the original domain restriction.
  4. Explain what the graph would show.
