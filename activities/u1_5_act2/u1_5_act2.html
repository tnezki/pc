Section 1.5 — StationsManistee High School
Section 1.5 Station 1 Solutions

Question 1

  1. As $x\to -1^-$, $f(x)\to 2$.
  2. As $x\to -1^+$, $f(x)\to 2$.
  3. Both one-sided limits are equal, so $\lim_{x\to -1} f(x)=2$.

Question 2

  1. $\lim_{x\to 2} f(x)=3$.
  2. $f(2)=-2$.
  3. The function value does not affect the limit because the graph approaches $3$ from both sides.

Question 3

A removable discontinuity can still have a two-sided limit because both sides approach the same value, even if the function value is missing or different.

A jump discontinuity has different left-hand and right-hand limits, so the two-sided limit does not exist.

At a vertical asymptote, the function grows without bound on at least one side, so there is no finite two-sided limit.

Question 4

  1. At $x=-3$: $\lim_{x\to -3^-}f(x)=1$, $\lim_{x\to -3^+}f(x)=1$, $f(-3)=-3$, so $\lim_{x\to -3}f(x)=1$.
  2. At $x=0$: $\lim_{x\to 0^-}f(x)=2$, $\lim_{x\to 0^+}f(x)=-1$, $f(0)=-1$, so $\lim_{x\to 0}f(x)$ does not exist.
  3. At $x=3$: $\lim_{x\to 3^-}f(x)=-1$, $\lim_{x\to 3^+}f(x)=1$, $f(3)=4$, so $\lim_{x\to 3}f(x)$ does not exist.

Question 10

The height of a ball is modeled by

\(h(t)=-16t^2+64t+5\)

where \(t\) is time in seconds and \(h(t)\) is height in feet.

  1. Find the time when the ball reaches its maximum height.

  2. Find the maximum height.

  3. What is the initial height?

  4. Explain what the vertex means in context.

Question 11

Compare the functions:

\(f(x)=x^2-4x+1\)

\(g(x)=-(x^2-4x+1)\)

  1. How are their axes of symmetry related?

  2. How are their vertices related?

  3. How are their maximum/minimum values related?

  4. Explain the effect of multiplying by \(-1\).

Question 12

A company sells \(400\) items when the price is \(\$20\). For each \(\$1\) increase in price, the company sells \(10\) fewer items.

  1. Write an expression for the price.

  2. Write an expression for the number of items sold.

  3. Write a revenue function.

  4. Find the price that maximizes revenue.

  5. Find the maximum revenue.

  6. Explain why the revenue model is quadratic.

Section 1.5 Station 3 Solutions

Question 9

  1. Axis of symmetry: $x=-\frac{4}{2(2)}=-1$.
  2. Vertex: $h(-1)=2(-1)^2+4(-1)-7=-9$, so $(-1,-9)$.
  3. $y$-intercept: $(0,-7)$.
  4. Domain: all real numbers.
  5. Range: $y\ge -9$.

Question 10

  1. The ball reaches its maximum at $t=-\frac{64}{2(-16)}=2$ seconds.
  2. The maximum height is $h(2)=69$ feet.
  3. The initial height is $h(0)=5$ feet.
  4. The vertex means the ball reaches $69$ feet after $2$ seconds.

Question 11

  1. Their axes of symmetry are the same: $x=2$.
  2. $f$ has vertex $(2,-3)$ and $g$ has vertex $(2,3)$.
  3. $f$ has a minimum value of $-3$; $g$ has a maximum value of $3$.
  4. Multiplying by $-1$ reflects the graph over the $x$-axis.

Question 12

  1. Let $x$ be the number of $1$ dollar increases. Price: $20+x$.
  2. Items sold: $400-10x$.
  3. $R(x)=(20+x)(400-10x)=-10x^2+200x+8000$.
  4. The maximum occurs at $x=-\frac{200}{2(-10)}=10$, so the price is $30$.
  5. The maximum revenue is $30(300)=9000$ dollars.
  6. The model is quadratic because it is the product of two linear expressions.

Question 16

\[B(x)=x+5\]

  1. Simplify \(A(x)\).

  2. State any excluded value for \(A(x)\).

  3. Compare \(A(x)\) and \(B(x)\) as expressions and as functions.

  4. Evaluate both at \(x=7\).

  5. Explain why restrictions matter when connecting algebra and graphs.

Section 1.5 Station 4 Solutions

Question 13

  1. $x+y=4+(-1)=3$
  2. $xy=4(-1)=-4$
  3. $x^2-y=16-(-1)=17$
  4. $2x-3y=8-3(-1)=11$

Question 14

  1. $f(x+1)=2(x+1)^2-(x+1)+3=2x^2+3x+4$
  2. $f(x-2)=2(x-2)^2-(x-2)+3=2x^2-9x+13$
  3. $f(2x)=2(2x)^2-2x+3=8x^2-2x+3$

Question 15

All three representations show the rule “multiply the input by $2$, then subtract $3$.” The table confirms this pattern because each output equals $2x-3$ for the listed input values.

Question 16

  1. $A(x)=\frac{(x-5)(x+5)}{x-5}=x+5$, when $x\ne 5$.
  2. The excluded value is $x=5$.
  3. As expressions after simplification they look the same, but as functions $A(x)$ has a hole at $x=5$ while $B(x)$ is defined there.
  4. $A(7)=12$ and $B(7)=12$.
  5. Restrictions matter because simplifying can hide holes or excluded values in a graph.
