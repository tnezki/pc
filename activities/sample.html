Problem 3

Compare these two tasks for $f(x)=x^2-4x$:

  1. Evaluate $f(6)$.
  2. Solve $f(x)=6$.

Complete both tasks and explain how they are different.

Section 1.1 — Rally CoachManistee High School
Solution 3
  1. $f(6)=6^2-4(6)=36-24=12$.
  2. $x^2-4x=6$, so $x^2-4x-6=0$.

Using the quadratic formula, $x=2\pm\sqrt{10}$.

Evaluating uses one input. Solving finds the input values that produce a given output.

Problem 9

A parabola has vertex $(3,7)$ and passes through the point $(5,15)$. Write the equation in vertex form.

Section 1.1 — Rally CoachManistee High School
Solution 9

Start with $y=a(x-3)^2+7$.

Use point $(5,15)$: $15=a(5-3)^2+7$.

$15=4a+7$, so $a=2$.

Answer: $y=2(x-3)^2+7$

Problem 11

A parabola has vertex $(1,6)$ and passes through $(3,-2)$. A student reasons $y=a(x+1)^2+6$, then uses $(3,-2)$ to find $a=-\frac{1}{2}$. Identify the error and solve correctly.

Section 1.1 — Rally CoachManistee High School
Solution 11

The student used $x+1$, which gives vertex $(-1,6)$ instead of $(1,6)$.

The correct form is $y=a(x-1)^2+6$.

Use $(3,-2)$: $-2=a(3-1)^2+6$.

$-8=4a$, so $a=-2$.

Answer: $y=-2(x-1)^2+6$

Problem 12

A bridge arch is modeled by a quadratic function with vertex $(0,30)$. The arch touches the ground at $(-10,0)$ and $(10,0)$.

  1. Write the equation in vertex form.
  2. Explain why the graph must open downward.
  3. Find the height of the arch when $x=5$.
  4. State the domain and range that make sense in context.
  5. Explain how the intercepts and vertex support your model.
Section 1.1 — Rally CoachManistee High School
Solution 12
  1. Use $y=ax^2+30$. Since $(10,0)$ is on the graph, $0=100a+30$, so $a=-\frac{3}{10}$. The model is $y=-\frac{3}{10}x^2+30$.
  2. The graph opens downward because the vertex is the highest point.
  3. $y=-\frac{3}{10}(5)^2+30=22.5$.
  4. Domain: $-10\le x\le 10$. Range: $0\le y\le 30$.
  5. The intercepts show where the arch reaches the ground, and the vertex shows the maximum height.

Problem 15

Compare the expressions:

$(3x)^2$ and $3x^2$

  1. Are they equivalent?
  2. Use $x=4$ to test both expressions.
  3. Explain the difference in structure.
Section 1.1 — Rally CoachManistee High School
Solution 15
  1. No, they are not equivalent.
  2. $(3\cdot4)^2=12^2=144$, but $3(4^2)=3\cdot16=48$.
  3. In $(3x)^2$, the exponent applies to both $3$ and $x$. In $3x^2$, the exponent applies only to $x$.

Problem 16

  1. Rewrite every factor with base $2$.
  2. Simplify to a single power of $2$.
  3. Explain why using the same base makes the expression easier to simplify.
Section 1.1 — Rally CoachManistee High School
Solution 16
  1. $2^x\cdot (2^3)^{x+1}\div(2^2)^{2x-1}$
  2. $2^x\cdot2^{3x+3}\div2^{4x-2}=2^{x+3x+3-(4x-2)}=2^5$
  3. Using the same base lets us add and subtract exponents.
