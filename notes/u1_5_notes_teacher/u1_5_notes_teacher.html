Section 1.5
Limits Graphically
Reading what a function approaches, not just what value it has.

Mathematical idea: A limit asks where the graph is heading. Sometimes the path matters more than the point.

In this section, you will use graphs to compare left-hand behavior, right-hand behavior, and actual function values near holes, jumps, and vertical asymptotes.

Learning Target

Students will analyze limits graphically by examining how functions behave as $x$ approaches a value from the left and right.

I can statements

  • I can determine left-hand and right-hand limits from a graph.
  • I can determine whether a limit exists using left-hand and right-hand limits near jumps, holes, and vertical asymptotes.
  • I can distinguish between the limit of a function and the actual function value shown on a graph.

Intro

Directions

Read the introduction aloud with a partner or in a small group. As you read, annotate the text by marking important ideas, circling key vocabulary, and writing short notes about connections, questions, or predictions in the margins.

A limit describes what value a function approaches as the input gets close to a specific number. The key word is approaches. When we ask for a limit, we are watching the graph move toward an $x$-value from nearby points, not simply looking at the point directly above that $x$-value.

Sometimes the limit and the function value are the same. This happens on many smooth, connected parts of graphs. But sometimes the graph has a hole, jump, or separate filled point. In those cases, the value the graph approaches may be different from the actual value of the function.

Limits are read from both sides. The left-hand limit describes what happens as $x$ approaches from values less than the target. The right-hand limit describes what happens as $x$ approaches from values greater than the target. A two-sided limit exists only when those two one-sided limits agree.

Graphical limits help us build language for behavior. Instead of only asking “What is the output?” we ask “What does the graph get close to?” That shift is one of the foundations for later calculus.

Stop and Discuss

  1. What does the word “approaches” mean when reading a graph?
  2. Why might a limit be different from a function value?
  3. What must be true about the left-hand and right-hand limits for the two-sided limit to exist?
  4. Which graph features from the reading might cause confusion?

Teacher Notes:

  1. “Approaches” means the graph gets close to a value as $x$ moves toward a target from nearby inputs.
  2. A limit can differ from a function value when there is a hole, jump, or separate filled point.
  3. The left-hand and right-hand limits must approach the same $y$-value.
  4. Likely confusion: filled point vs. open circle, jumps, holes, and asymptotes.

Misconception to watch: Students may point to the filled dot automatically. Push them to trace the graph from both sides.

Limits Describe Approaching Behavior

The notation $\lim_{x\to a} f(x)$ asks what value $f(x)$ approaches as $x$ gets close to $a$.

Graphical Limit: Follow the graph toward the target $x$-value from nearby inputs. The limit is the $y$-value the graph approaches, if both sides approach the same value.

Example 1

Use the graph to determine $\lim_{x\to 1} f(x)$.

Teacher Notes:

From the graph, as $x o 1$, the curve approaches $y=-2$ from both sides.

Answer: $\lim_{x o 1} f(x)=-2$.

Facilitation: Emphasize that this example is intentionally smooth, so the limit and visible function value agree.

You Try It 1

On a smooth connected graph, $f(x)$ approaches $4$ as $x$ approaches $-3$. What is $\lim_{x\to -3}f(x)$? Explain how you know.

Teacher Notes:

Answer: $\lim_{x o -3} f(x)=4$.

The graph approaches the same output, $4$, as $x$ gets close to $-3$.

Stop and Discuss

  1. Why do we look near $x=1$ instead of only at $x=1$?
  2. How can a graph show a limit even before you know an equation?
  3. When does the function value seem to match the limit?

Teacher Notes:

  1. Because the limit is about behavior near $x=1$, not only the single point at $x=1$.
  2. A graph shows what $y$-value the curve gets close to as inputs approach the target.
  3. On a smooth connected graph with no hole or jump, the limit and function value usually match.

Left-Hand and Right-Hand Limits

A one-sided limit tells what the graph approaches from only one side of the target input.

Left-hand limit: $\lim_{x\to a^-} f(x)$ follows the graph from the left.

Right-hand limit: $\lim_{x\to a^+} f(x)$ follows the graph from the right.

Example 2

Use the graph to determine $\lim_{x\to 0^-}f(x)$, $\lim_{x\to 0^+}f(x)$, and whether $\lim_{x\to 0}f(x)$ exists.

Teacher Notes:

From the left, the graph approaches the open point at $y=1$. From the right, it starts at/approaches $y=3$.

Answers: $\lim_{x o 0^-}f(x)=1$, $\lim_{x o 0^+}f(x)=3$, so $\lim_{x o 0}f(x)$ does not exist.

Misconception: The filled point at $(0,3)$ gives $f(0)$, but it does not make the two-sided limit exist.

You Try It 2

A graph approaches $-2$ from the left as $x\to 5$ and approaches $6$ from the right as $x\to 5$. Does $\lim_{x\to 5}f(x)$ exist? Explain.

Teacher Notes:

Answer: No. The two-sided limit does not exist because the one-sided limits are different: left side $=-2$, right side $=6$.

Limits and Function Values Can Be Different

A hole or separate filled point can make the limit and function value different. The limit depends on where the graph is heading. The function value depends on the filled point.

Example 3

Use the graph to determine $\lim_{x\to 2}f(x)$ and $f(2)$. Are they equal?

Teacher Notes:

The graph approaches the hole at about $y=2.2$, while the filled point shows $f(2)=-1.5$.

Answer: $\lim_{x o 2}f(x)pprox 2.2$ and $f(2)=-1.5$. They are not equal.

Teacher note: Accept an approximate limit from the graph. The key idea is that the limit comes from the line, not the filled point.

You Try It 3

A graph has a hole at $(4,2)$ and a closed point at $(4,-1)$. What are $\lim_{x\to 4}f(x)$ and $f(4)$?

Teacher Notes:

Answer: $\lim_{x o 4}f(x)=2$ and $f(4)=-1$.

The hole shows the approached value; the closed point shows the actual function value.

Stop and Discuss

  1. Which mark on the graph shows the function value?
  2. Which part of the graph shows the limit?
  3. Why can a graph have a hole and still have a limit?

Teacher Notes:

  1. The filled point shows the actual function value.
  2. The nearby curve or path of the graph shows the limit.
  3. A hole can still have a limit because both sides may approach the same missing point.

Teacher move: Ask students to cover the filled point and trace the curve first.

When a Limit Does Not Exist

A two-sided limit does not exist when the left-hand and right-hand limits approach different values, or when the graph grows without bound instead of approaching one finite number.

Example 4

Use the graph to describe the behavior as $x\to 0$. Does $\lim_{x\to 0}f(x)$ exist as a finite number?

Teacher Notes:

As $x o 0^-$, the graph decreases without bound. As $x o 0^+$, the graph increases without bound.

Answer: The two-sided limit does not exist as a finite number.

Language cue: Students may say “undefined.” Push for a graph-based explanation: the function grows without bound near the vertical asymptote.

You Try It 4

A graph approaches $+\infty$ from the left and $-\infty$ from the right as $x\to -3$. Describe why the two-sided limit does not exist as a finite number.

Teacher Notes:

Answer: The two-sided limit does not exist as a finite number because the graph is unbounded and the two sides do not approach the same behavior.

Stop and Discuss

  1. How is an infinite limit different from a finite limit?
  2. Why is “does not exist” not always the same as “nothing is happening”?
  3. What graph features should make you check one-sided limits carefully?

Teacher Notes:

  1. An infinite limit describes unbounded behavior, while a finite limit approaches one real-number output.
  2. “Does not exist” can mean the graph has behavior, but it is not approaching one finite value.
  3. Check one-sided limits carefully near jumps, holes, open circles, separate filled points, and vertical asymptotes.

Reasoning Across Multiple Graph Features

Many graphs combine holes, jumps, point values, and connected pieces. When reading these graphs, slow down and check each target value one side at a time.

Example 5

Use the graph to answer each question.

  1. Determine $\lim_{x\to -2^-} f(x)$ and $\lim_{x\to -2^+} f(x)$.
  2. Determine whether $\lim_{x\to -2} f(x)$ exists.
  3. Determine $f(-2)$.
  4. Determine whether $\lim_{x\to 2} f(x)$ exists.

Teacher Notes:

  1. $\lim_{x o -2^-}f(x)=2$ and $\lim_{x o -2^+}f(x)=-1$.
  2. $\lim_{x o -2}f(x)$ does not exist because the one-sided limits are different.
  3. $f(-2)=-2$ from the filled point.
  4. $\lim_{x o 2}f(x)$ does not exist because the left side approaches $-1$ and the right side approaches $3$.

Teacher note: This is the best place to insist on three separate questions: left behavior, right behavior, and actual function value.

You Try It 5

Create a quick sketch of a graph where $\lim_{x\to 1}f(x)=3$ but $f(1)=-2$. Explain which part of your sketch shows the limit and which part shows the function value.

Teacher Notes:

Sample answer: Draw a curve approaching an open circle at $(1,3)$ from both sides, then draw a filled point at $(1,-2)$.

The open-circle path shows $\lim_{x o 1}f(x)=3$; the filled point shows $f(1)=-2$.

Summary

  • A limit describes what $f(x)$ approaches as $x$ gets close to a target input.
  • Left-hand limits come from values less than the target input.
  • Right-hand limits come from values greater than the target input.
  • A two-sided limit exists only when the left-hand and right-hand limits agree.
  • A function value is shown by the filled point at the input.
  • A limit can exist even if the function value is different or undefined.
  • Jumps and vertical asymptotes often cause two-sided limits not to exist.

Teacher Notes:

Closure prompt: Ask students to write one sentence using all three phrases: left-hand limit, right-hand limit, function value.

Common Mistakes

  • Using the filled point for every answer: the filled point gives $f(a)$, not always $\lim_{x\to a}f(x)$.
  • Ignoring one side: a two-sided limit requires both one-sided limits to agree.
  • Confusing a hole with no limit: a hole may still have a limit if both sides approach the same value.
  • Saying a jump has one limit: if the left and right sides approach different values, the two-sided limit does not exist.
  • Writing infinity as a normal output: infinite behavior describes unbounded growth, not a finite function value.
  • Reading the wrong coordinate: limits are reported as approached $y$-values, not the target $x$-value.
