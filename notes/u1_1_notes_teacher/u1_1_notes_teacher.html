Section 1.1
Function Notation & Solving
Teacher version with inline answers, discussion notes, and facilitation guidance.
Evaluating functions, solving $f(x)=a$, and connecting algebra, tables, and graphs.

Mathematical idea: A function is more than an equation. It is a relationship that lets us ask two different questions: What output comes from this input? What input creates this output?

In this section, you will learn how function notation organizes inputs and outputs, how solving differs from evaluating, and why graphs and tables can confirm algebraic work.

Learning Target

Students will evaluate functions and solve equations of the form $f(x) = a$ using tables, algebraic and graphical methods.

I can statements

  • I can evaluate a function written in function notation for given input values.
  • I can solve equations of the form $f(x)=a$ algebraically when $f(x)$ is linear or a factorable quadratic.
  • I can solve equations of the form $f(x)=a$ graphically or with a table.
  • I can verify and interpret solutions within the context of the function and its graph.

A school sells tickets to a fundraiser. The profit, in dollars, can be modeled by

\[ P(x)=-2(x-10)(x-40) \]

where $x$ represents the ticket price in dollars.

  1. Evaluate $P(20)$ and interpret its meaning.

Intro

Directions

Read the introduction aloud with a partner or in a small group. As you read, annotate the text by marking important ideas, circling key vocabulary, and writing short notes about connections, questions, or predictions in the margins.

Function notation is a compact way to describe a relationship between inputs and outputs. When we write $f(x)$, we are not multiplying $f$ and $x$. Instead, $f(x)$ means “the output of the function $f$ when the input is $x$.” This notation lets us name a rule, substitute values into that rule, and talk precisely about how one quantity depends on another.

Evaluating a function means starting with a known input and finding the output. For example, if $f(x)=2x+3$, then $f(4)$ asks for the output when the input is $4$. Solving an equation such as $f(x)=11$ asks a different question: which input values make the output equal to $11$?

This distinction matters throughout precalculus. A table can show which inputs match a specific output. A graph can show solutions as intersection points between the function and a horizontal line. Algebra can give exact answers when the equation is linear or factorable. Each representation tells the same story in a different way.

Function notation is also used in science, engineering, business, and computer science. If $H(t)$ represents height after $t$ seconds, then $H(3)$ is the height at 3 seconds. If $C(n)$ represents cost for $n$ items, then solving $C(n)=100$ asks how many items create a cost of $100$. The notation helps connect mathematics to decisions and interpretations.

Stop and Discuss

  1. What does $f(x)$ mean in words?
  2. How is evaluating $f(4)$ different from solving $f(x)=4$?
  3. What does a graph show when solving $f(x)=a$?
  4. What is one real-world example of function notation from the text?

Sample Responses: $f(x)$ means the output of function $f$ for input $x$. Evaluating $f(4)$ starts with input $4$; solving $f(x)=4$ looks for input(s) that produce output $4$. A graph shows solutions as intersections with the horizontal line $y=a$. Real-world examples include height after time, cost for items, or pages left after time.

Function Notation Means Input and Output

A function notation statement has three important pieces.

Notation Reminder

  • $f$ is the name of the function.
  • $x$ is the input variable.
  • $f(x)$ is the output value.

When you see $f(3)$, replace every $x$ in the rule with $3$.

Example 1

Let $f(x)=2x^2-3$. Evaluate $f(2)$ and $f(-3)$.

Answer:

\[ f(2)=2(2)^2-3=8-3=5 \] \[ f(-3)=2(-3)^2-3=18-3=15 \]

You Try It 1

Let $g(x)=x^2-2x$. Evaluate $g(3)$ and $g(-2)$.

Answer: $g(3)=3$ and $g(-2)=8$.

Stop and Discuss

  1. Why are parentheses important when substituting a negative input?
  2. What does the answer to $f(2)$ represent?
  3. Why does $f(-3)$ not mean the opposite of $f(3)$?

Sample Responses: Parentheses preserve the entire input, especially negative inputs. $f(2)$ represents the output when the input is $2$. In general, $f(-3)$ is not the opposite of $f(3)$ because the rule may include powers, products, or other operations that do not create opposite outputs.

Substitution and Order of Operations

Function evaluation depends on careful substitution. The input goes inside parentheses first, then the operations are completed in the correct order.

Careful Substitution

\[ f(-2)=3(-2)^2-4(-2)+1 \]

The exponent applies to the entire substituted input because the input is inside parentheses.

Example 2

A student evaluates $f(-2)$ for $f(x)=3x^2-4x+1$ and gets $-3$.

  1. Identify the likely mistake.
  2. Find the correct value of $f(-2)$.
  3. Explain how parentheses affect evaluation.

Answer: The likely mistake is treating $(-2)^2$ as $-4$ or dropping parentheses.

\[ f(-2)=3(-2)^2-4(-2)+1=12+8+1=21 \]

Parentheses show that the entire input is squared.

You Try It 2

Let $p(x)=-3x^2+1$. Evaluate $p(1)$ and $p(-2)$.

Answer: $p(1)=-2$ and $p(-2)=-11$.

Evaluating vs. Solving

One of the most important distinctions in this section is the difference between evaluating and solving.

Evaluate: You are given the input and you find the output.

Solve: You are given the output and you find the input or inputs.

TaskQuestion Being Asked
$f(6)$What output happens when the input is $6$?
$f(x)=6$What input or inputs make the output $6$?

Example 3

Compare these two tasks for $f(x)=x^2-4x$:

  1. Evaluate $f(6)$.
  2. Solve $f(x)=6$.
  3. Explain how they are different.

Answer:

\[ f(6)=6^2-4(6)=36-24=12 \]

To solve $f(x)=6$:

\[ x^2-4x=6 \quad \Rightarrow \quad x^2-4x-6=0 \] \[ x=2\pm\sqrt{10} \]

Evaluating starts with a known input. Solving starts with a known output and finds the input values.

You Try It 3

Let $f(x)=2(x-1)(x+4)$.

  1. Solve $f(x)=0$.
  2. Evaluate $f(0)$.
  3. Explain the difference between solving $f(x)=0$ and evaluating $f(0)$.

Answer: $f(x)=0$ when $x=1$ or $x=-4$. Also $f(0)=2(-1)(4)=-8$. Solving finds input values that make the output $0$; evaluating $f(0)$ finds the output for input $0$.

Stop and Discuss

A classmate says, “Solving $f(x)=a$ is the same thing as evaluating $f(a)$.” What is wrong with this claim?

Sample Answer: Solving $f(x)=a$ means the output is known and the input is unknown. Evaluating $f(a)$ means the input is known and the output is unknown. These are inverse types of questions, not the same task.

Solving $f(x)=a$ with a Table

When a function is represented by a table, solving $f(x)=a$ means finding the input value or values where the output row equals $a$.

Example 4

Use the table to solve $q(x)=6$.

$x$-4-2024
$q(x)$1462614

Answer: $q(x)=6$ when $x=-2$ and $x=2$.

You Try It 4

A function $f$ is represented by the table below.

$x$-2-10123
$f(x)$-3010-3-8
  1. Solve $f(x)=0$ using the table.
  2. Describe what those solutions would look like on a graph.

Answer: $f(x)=0$ when $x=-1$ and $x=1$. On a graph, those would be x-intercepts at $(-1,0)$ and $(1,0)$.

Stop and Discuss

  1. Can a table show more than one solution to $f(x)=a$?
  2. Can a table prove that there are no other solutions? Why or why not?
  3. How is looking across a table different from substituting one value?

Sample Responses: Yes, a table can show more than one listed input with the same output. A table usually cannot prove there are no other solutions unless it represents the entire domain or enough structure is known. Looking across a table compares many inputs at once, while substituting one value only checks one input.

Solving $f(x)=a$ Algebraically

When the function rule is known, algebra can often give exact solutions. In this section, the algebraic solving will focus on linear functions and factorable quadratics.

Linear: isolate the variable.

Factorable quadratic: set the equation equal to zero, factor, and use the zero product property.

Example 5

Let $g(x)=2x+1$.

  1. Solve $g(x)=9$ algebraically.
  2. Create a two-row table that confirms your solution.

Answer: $2x+1=9$, so $2x=8$ and $x=4$. A confirming table can include $x=4$ with $g(4)=9$.

You Try It 5

Let $k(x)=x^2+2x-8$.

  1. Solve $k(x)=0$ by factoring.
  2. State the ordered pairs where the graph crosses the $x$-axis.

Answer: $x^2+2x-8=(x+4)(x-2)$, so $x=-4$ or $x=2$. The graph crosses the x-axis at $(-4,0)$ and $(2,0)$.

Graphs Show Solutions as Intersections

Solving $f(x)=a$ graphically means finding where the graph of $y=f(x)$ intersects the horizontal line $y=a$.

Graphical Meaning

The solution to $f(x)=a$ is the $x$-coordinate of each intersection between the function and the horizontal line $y=a$.

Graph Task

The graph of $f(x)=2x-3$ and the horizontal line $y=5$ are shown. Use the graph to solve $f(x)=5$, then verify algebraically.

Answer: The graph intersects at $(4,5)$, so $x=4$. Algebra check: $2x-3=5 \Rightarrow 2x=8 \Rightarrow x=4$.

Partner Task

The graph of $g(x)=x^2-4$ and the horizontal line $y=5$ are shown. Use the graph to solve $g(x)=5$, then verify algebraically.

Answer: The graph shows $x=-3$ and $x=3$. Algebra check: $x^2-4=5 \Rightarrow x^2=9 \Rightarrow x=\pm3$.

Stop and Discuss

  1. Why is a horizontal line used when solving $f(x)=a$?
  2. Why can a quadratic equation have two solutions for the same output?
  3. What does the ordered pair at an intersection mean?

Sample Responses: A horizontal line is used because $f(x)=a$ means the output is fixed at $a$. A quadratic can have two inputs that produce the same output because of symmetry. The ordered pair gives both the input and output; the solution is the $x$-coordinate.

Interpreting Solutions in Context

In a context, the input and output have meaning. A solution may be mathematically correct but still need to be interpreted carefully.

Context Task

Let $P(t)=25-5t$, where $P(t)$ is the number of pages left to read after $t$ minutes.

  1. Find $P(3)$.
  2. Solve $P(t)=0$.
  3. Interpret both answers in context.

Answer: $P(3)=25-15=10$, so 10 pages remain after 3 minutes. $P(t)=0$ gives $25-5t=0$, so $t=5$. The reader finishes after 5 minutes.

Partner Context Task

The graph models the height $A(t)$, in meters, of an object after $t$ seconds.

  1. Estimate the times when $A(t)=16$.
  2. Explain why both solutions make sense in the context.
  3. Describe a time value that would not make sense in this context and explain why.

Answer: The graph shows $A(t)=16$ at approximately $t=0$ and $t=6$. Both make sense because the object is at that height on the way up and on the way down. Negative time values would not make sense in this context.

Reasoning and Error Analysis

Error analysis helps reveal the structure behind the procedure. When a solution is wrong, ask whether the student evaluated instead of solved, dropped a solution, substituted incorrectly, or misread the graph.

Error Analysis Task

Let $f(x)=x^2-9$. A student says the solution to $f(x)=0$ is $x=3$. Determine whether the student is correct and explain.

Answer: The student is only partially correct. Since $x^2-9=(x-3)(x+3)$, the solutions are $x=3$ and $x=-3$. The student missed one solution.

Partner Error Analysis

Two students solve $f(x)=0$ for $f(x)=x^2-7x+10$.

Student A: $(x-5)(x-2)=0$, so $x=5$ or $x=2$.

Student B: $x^2=7x-10$, so $x=7$ or $x=-10$.

Which student is correct? Explain the error in the other solution.

Answer: Student A is correct. Student B incorrectly split $x^2=7x-10$ into two separate equations or guessed values without factoring. Correct solutions are $x=5$ and $x=2$.

Stop and Discuss

  1. Why is it easy to miss one solution when solving a quadratic?
  2. How can substitution verify a solution?
  3. How can a graph help catch an algebra mistake?

Sample Responses: Quadratics often have two solutions because the graph may cross a horizontal line twice. Substitution verifies a solution by checking whether it makes the equation true. A graph helps reveal missing solutions or unreasonable algebraic answers.

Return to the Future Task

Use what you learned to revisit the task from the beginning of the notes.

A school sells tickets to a fundraiser. The profit, in dollars, can be modeled by

\[ P(x)=-2(x-10)(x-40) \]

where $x$ represents the ticket price in dollars.

  1. Evaluate $P(20)$ and interpret its meaning.
  2. Solve $P(x)=0$ and interpret the solutions in context.
  3. Solve $P(x)=600$ using a graph or table, then verify algebraically.
  4. Explain why evaluating $P(20)$ and solving $P(x)=20$ are different tasks.

Answer Guide:

  1. $P(20)=-2(10)(-20)=400$, so a $20$ ticket price gives a profit of $400$.
  2. $P(x)=0$ when $x=10$ or $x=40$; these ticket prices break even.
  3. $-2(x-10)(x-40)=600$ gives $x^2-50x+700=0$. The discriminant is $2500-2800=-300$, so there are no real ticket prices that produce exactly $600$ profit with this model.
  4. Evaluating $P(20)$ uses the input $20$ to find profit. Solving $P(x)=20$ would find ticket prices that produce a profit of $20$.

Summary

  • Function notation connects inputs to outputs.
  • Evaluating $f(a)$ means using a known input to find an output.
  • Solving $f(x)=a$ means finding the input value or values that produce a given output.
  • Tables solve $f(x)=a$ by locating output values equal to $a$.
  • Graphs solve $f(x)=a$ by showing intersections with the horizontal line $y=a$.
  • Algebra can verify or find exact solutions.
  • Context determines what inputs and outputs mean and whether solutions are reasonable.

Common Mistakes

  • Confusing $f(a)$ with $f(x)=a$: one asks for an output, the other asks for input(s).
  • Forgetting parentheses with negative inputs: $(-2)^2$ and $-2^2$ are not the same.
  • Finding only one quadratic solution: a quadratic can have two inputs with the same output.
  • Reading the wrong coordinate from a graph: when solving $f(x)=a$, the answer is the $x$-value of the intersection.
  • Not checking context: some mathematical solutions may not make sense in the situation.
  • Using a table too broadly: a table shows listed values, but it may not show every possible solution.
