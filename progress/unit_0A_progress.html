Unit A Practice by Depth of Knowledge

DoK 1 — Rungs 1–2 Students can recognize quadratic structure, identify basic graph features, use vocabulary accurately, and distinguish between different quadratic forms.
1.

Identify the form of each quadratic equation: standard form, vertex form, or intercept form.

  1. \(y=2x^2-5x+3\)

  2. \(y=-3(x-4)^2+7\)

  3. \(y=(x+2)(x-6)\)

  4. \(y=-\frac12(x-1)(x+5)\)

Workspace
2.

Identify the vertex of each quadratic.

  1. \(y=(x-3)^2+5\)

  2. \(y=(x+4)^2-2\)

  3. \(y=-2(x-1)^2+7\)

Workspace
3.

Match each function to its transformation from \(y=x^2\).

  1. \(y=(x-4)^2\)

  2. \(y=x^2+6\)

  3. \(y=-x^2\)

  4. \(y=3x^2\)

Transformations:

  1. reflection over the \(x\)-axis

  2. shift right 4

  3. vertical stretch by a factor of 3

  4. shift up 6

Workspace
4.

For each function, state the \(x\)-intercepts.

  1. \(f(x)=(x-4)(x+2)\)

  2. \(g(x)=3(x-1)(x-8)\)

  3. \(h(x)=-2(x+6)(x-3)\)

Workspace
5.

Use the graph to identify the key features of the quadratic function.

  1. Identify the vertex.

  2. State the axis of symmetry.

  3. Identify the \(x\)-intercepts.

  4. State whether the vertex is a maximum or minimum.

  5. Write a possible equation in vertex form.

Workspace
6.

For each value of \(\Delta\), state the number of real solutions.

  1. \(\Delta=25\)

  2. \(\Delta=0\)

  3. \(\Delta=-12\)

  4. \(\Delta=3\)

Workspace
7.

Use the graph to write and interpret a quadratic model.

  1. Identify the vertex and explain what it tells you about the graph.

  2. Identify the \(x\)-intercepts.

  3. State the axis of symmetry.

  4. Write a possible equation in intercept form.

  5. Rewrite the equation in vertex form.

Workspace
8.

Match each description to the correct range.

  1. A parabola opens up and has vertex \((3,-4)\).

  2. A parabola opens down and has vertex \((-2,7)\).

  3. A parabola opens up and has vertex \((0,5)\).

  4. A parabola opens down and has vertex \((6,-1)\).

Ranges:

  1. \(y\le 7\)

  2. \(y\ge -4\)

  3. \(y\le -1\)

  4. \(y\ge 5\)

A quadratic has \(x\)-intercepts at \((-4,0)\) and \((6,0)\), and passes through \((0,-24)\).

  1. Find the first differences.

  2. Find the second differences.

  3. Determine the value of \(a\).

find:

  3. the vertex

  4. whether the vertex is a maximum or minimum

  5. the range

\(h(t)=-16t^2+64t+5\)

where \(t\) is time in seconds and \(h(t)\) is height in feet.

  1. Find the time when the ball reaches its maximum height.

  2. Find the maximum height.

  3. What is the initial height?

  4. Explain what the vertex means in context.

DoK 3 — Rungs 5–6 Students can reason strategically, justify conclusions, connect representations, explain structure, and interpret solutions and key features.
1.

A student says, “The graph of \(y=(x+5)^2-2\) shifts right 5 and down 2.” Explain the student’s mistake. Then describe the correct transformation.

Workspace
2.

A student factors:

\(x^2+2x-24\)

as:

\((x+6)(x-4)\)

Then solves:

\(x=-6,\quad x=4\)

Is the student correct? Explain how you can check without expanding everything.

Workspace
3.

A student calculates the discriminant for:

\[2x^2-3x+8=0\]

as:

\[\Delta=(-3)^2-4(2)(8)=9-64=-55\]

“Since the discriminant is negative, there are no solutions.”

What should the student have said instead? Explain carefully.

Workspace
4.

Compare these two equations: \[ f(x)=2(x-3)^2-8 \] \[ g(x)=2(x-1)(x-5) \]

  1. Rewrite both equations in standard form.

  2. Explain why they represent the same function.

  3. State the vertex and zeros.

Workspace
5.

The table shows a quadratic function. \[ \begin{array}{c|ccccc} x&-3&-2&-1&0&1\\ \hline y&14&5&0&-1&2 \end{array} \]

  1. Use differences to verify the function is quadratic.

  2. Write the equation in standard form.

  3. Rewrite the equation in vertex form.

  4. Explain what each form reveals.

Workspace
6.

A student says:

“The range of \(y=-3(x-2)^2+5\) is \(y\ge 5\) because the vertex has \(y=5\).”

Explain the mistake and state the correct range.

Workspace
7.

A quadratic has \(x\)-intercepts at \((-1,0)\) and \((7,0)\), and opens upward.

  1. What is the axis of symmetry?

  2. What kind of value does the vertex represent?

  3. Where is the function decreasing?

  4. Where is the function increasing?

  5. Explain how you know without finding the full equation.

Workspace
8.

For each equation, choose the most efficient method: factoring, completing the square, or quadratic formula. Then justify your choice.

  1. \(x^2-9x+20=0\)

  2. \(x^2+4x-1=0\)

  3. \(3x^2-8x+2=0\)

  4. \(x^2+12x+36=0\)

DoK 4 — Rungs 7–8 Students can synthesize solving, graphing, equation writing, rewriting forms, and interpretation to model and analyze unfamiliar quadratic situations.
1.

A water fountain shoots water upward. The height of the water is modeled by \(h(t)=-5(t-2)^2+21\), where \(t\) is time in seconds and \(h(t)\) is height in feet.

  1. Identify the vertex and interpret it in context.

  2. Determine the height at \(t=0\).

  3. Determine another time when the water is the same height as at \(t=0\).

  4. Explain how symmetry helps answer part c.

  5. State a reasonable domain for the model and justify it.

Workspace
2.

A ball is launched from the ground. Its height is modeled by:

\(h(t)=-2t(t-6)\)

where \(t\) is time in seconds and \(h(t)\) is height in meters.

  1. What are the zeros of the function?

  2. Interpret each zero in context.

  3. Find the time when the ball reaches its maximum height.

  4. Find the maximum height.

  5. Explain how factored form helps answer the question efficiently.

Workspace
3.

A ball is thrown upward from a platform. Its height is modeled by:

\[h(t)=-16t^2+48t+64\]

where \(h(t)\) is height in feet and \(t\) is time in seconds.

  1. Rewrite the function in vertex form by completing the square.

  2. Identify and interpret the vertex.

  3. Use the quadratic formula to determine when the ball hits the ground.

  4. Explain why one solution may not make sense in context.

  5. State a reasonable domain for the model.

Workspace
4.

A quadratic function is shown by the table. \[ \begin{array}{c|ccccc} x&0&1&2&3&4\\ \hline f(x)&12&3&-2&-3&0 \end{array} \]

  1. Use first and second differences to verify the function is quadratic.

  2. Write the equation in standard form.

  3. Rewrite the equation in vertex form.

  4. Find the zeros using a method of your choice.

  5. Explain how the table, standard form, vertex form, and zeros all describe the same function.

Workspace
5.

A farmer has \(100\) meters of fencing to enclose a rectangular field along a river. No fencing is needed along the river. Let \(x\) be the width perpendicular to the river.

  1. Write an area function in terms of \(x\).

  2. Find the dimensions that maximize area.

  3. Find the maximum area.

  4. State a reasonable domain for the model.

  5. Explain why the domain is restricted in context.

Workspace
6.

A quadratic has zeros \(x=-4\) and \(x=10\). It passes through \((2,-48)\).

  1. Write the equation in intercept form.

  2. Find the vertex.

  3. Rewrite the equation in vertex form.

  4. Rewrite the equation in standard form.

  5. Explain which form is most useful for finding zeros, vertex, and \(y\)-intercept.

Workspace
7.

Two different quadratic models are proposed for the same revenue situation:

\(R_1(p)=-2(p-12)^2+288\)

\(R_2(p)=-2p^2+48p\)

  1. Show that the two models are equivalent.

  2. Identify the price that maximizes revenue.

  3. Identify the maximum revenue.

  4. Find the break-even prices.

  5. Explain which form is most useful for each feature.

Workspace
8.

Two students solve the same equation:

\[2x^2-8x-3=0\]

Student A uses completing the square.
Student B uses the quadratic formula.

  1. Solve the equation using completing the square.

  2. Solve the equation using the quadratic formula.

  3. Show that the answers are equivalent.

  4. Explain which method is more efficient for this equation and why.
