Unit 2 Practice by Depth of Knowledge

DoK 1 — Rungs 1–2 Students can recognize functions from equations, graphs, tables, and descriptions; evaluate simple function notation; and identify core vocabulary for symmetry, function operations, composition, average rate of change, and inverses.
1.

A graph is shown. Use symmetry to classify the function as even, odd, or neither.

  1. What type of symmetry do you see?
  2. What should be true about \(f(-x)\)?
Workspace
2.

Use the table to determine whether the function appears even, odd, or neither.

\(x\)\(-2\)\(-1\)\(1\)\(2\)
\(f(x)\)\(4\)\(1\)\(1\)\(4\)

Justify using the table.

Workspace
3.

The graphs of \(f\) and \(g\) are shown.

  1. Estimate \(f(2)\) and \(g(2)\).
  2. Estimate \((f+g)(2)\).
  3. Estimate \((f-g)(2)\).
Workspace
4.

Use the table to evaluate each expression.

\(x\)\(-2\)\(0\)\(3\)
\(f(x)\)\(5\)\(1\)\(-4\)
\(g(x)\)\(-2\)\(6\)\(3\)
  1. \((f+g)(0)\)
  2. \((fg)(-2)\)
  3. \(\left(\frac{f}{g}\right)(3)\)
Workspace
5.

Use the table.

\(x\)\(-2\)\(-1\)\(0\)\(1\)\(2\)
\(f(x)\)\(1\)\(3\)\(2\)\(-1\)\(0\)
\(g(x)\)\(2\)\(0\)\(-1\)\(3\)\(1\)
  1. \(f(g(0))\)
  2. \(g(f(2))\)
  3. \(f(f(1))\)
Workspace
6.

Decide whether each expression represents function composition or function arithmetic.

  1. \(f(g(x))\)
  2. \(f(x)+g(x)\)
  3. \((f\circ g)(x)\)
  4. \(f(x)g(x)\)
Workspace
7.

Use the table to find the average rate of change from \(x=0\) to \(x=3\).

\(x\)\(0\)\(1\)\(2\)\(3\)
\(f(x)\)\(2\)\(5\)\(10\)\(17\)
Workspace
8.

Use the horizontal line test to decide whether each graph has an inverse that is a function.

DoK 2 — Rungs 3–4 Students can evaluate function operations, compositions, inverse expressions, and average rates of change from multiple representations, then calculate and simplify function arithmetic, composite functions, difference quotients, and inverse functions while identifying restrictions when needed.
1.

The graph of \(f\) and the graph of \(g\) are shown separately.

  1. Estimate \(f(1)\) and \(g(1)\), then find \((f+g)(1)\).
  2. Estimate \(f(4)\) and \(g(4)\), then find \((f-g)(4)\).
  3. Use your graph values to find \((fg)(1)\).
  4. Explain how you used the two graphs.
Workspace
2.

Use the graph of \(f\) and the table for \(g\).

\(x\)\(-2\)\(-1\)\(0\)\(1\)\(2\)
\(g(x)\)\(1\)\(3\)\(-1\)\(2\)\(0\)
  1. Find \(g(0)\).
  2. Use the graph of \(f\) to find \(f(g(0))\).
  3. Find \(g(2)\).
  4. Use the graph of \(f\) to find \(f(g(2))\).
  5. Explain how the table output becomes the graph input.
Workspace
3.

Use the equation for \(f\) and the graph of \(g\).

\(f(x)=2x-1\)

  1. Estimate \(g(3)\) from the graph.
  2. Find \(f(g(3))\).
  3. Estimate \(g(-1)\) from the graph.
  4. Find \(f(g(-1))\).
  5. Explain why the graph value must be found before using the equation.
Workspace
4.

Use a graph of \(f(x)=x^2-2\) to estimate the average rate of change from \(x=1\) to \(x=3\). Explain how this matches the idea of the difference quotient.

Workspace
5.

For \(f(x)=\sqrt{x}\), use a graph to estimate the average rate of change from \(x=1\) to \(x=4\). Then verify using exact function values.

Workspace
6.

Use the graph of \(f\).

  1. Sketch \(f^{-1}\).
  2. Identify two reflected point pairs.
  3. State whether \(f^{-1}\) is a function.
Workspace
7.

Let \(f(x)=\sqrt{x}\) and \(g(x)=x-5\).

  1. Find \(f(g(x))\).
  2. State the domain of \(f(g(x))\).
  3. Explain why the domain is restricted.
Workspace
8.

Verify that \(f(x)=2x-1\) and \(g(x)=\frac{x+1}{2}\) are inverses using composition.

DoK 3 — Rungs 5–6 Students can justify conclusions about symmetry, domain restrictions, composition, average rate of change, and inverse relationships while explaining how function structure changes across representations.
1.

Consider the piecewise-defined function.

\(f(x)=\begin{cases}x^2 & x\ge 0\\ -x^2 & x<0\end{cases}\)

  1. Classify the function as even, odd, or neither.
  2. Justify algebraically.
  3. Describe its symmetry.
Workspace
2.

Let \(f(x)=\sqrt{x+1}\) and \(g(x)=6-x\).

  1. Find the domain of \(f+g\).
  2. Find the domain of \(fg\).
  3. Find the domain of \(\frac{f}{g}\).
  4. Explain why the quotient has one additional restriction.
Workspace
3.

Use the graph of \(f\) and the graph of \(g\).

  1. Estimate \(g(1)\), then use that value to estimate \(f(g(1))\).
  2. Estimate \(f(1)\), then use that value to estimate \(g(f(1))\).
  3. Are the results the same?
  4. Explain why the order of composition matters using the graphs.
Workspace
4.

Use graph values to estimate average rate of change from \(x=-2\) to \(x=2\). Then explain why another interval with the same length may have a different average rate of change.

Workspace
5.

A cyclist’s distance is modeled by \(d(t)=4t^2+8t\).

  1. Find the average speed from \(t=1\) to \(t=4\).
  2. Find the average speed from \(t=4\) to \(t=7\).
  3. Explain why the two average speeds are different.
Workspace
6.

A graph is reflected across \(y=x\). Explain how:

  1. domain and range change
  2. \(x\)- and \(y\)-intercepts change
  3. points on the graph change
Workspace
7.

Compare function arithmetic and function composition.

Use examples to explain the difference between \((f+g)(x)\) and \(f(g(x))\).

Your explanation should include meaning, notation, and order.

Workspace
8.

A student finds the inverse of \(f(x)=\sqrt{x+6}\) as \(f^{-1}(x)=x^2+6\).

  1. Identify the error.
  2. Find the correct inverse.
  3. State the correct domain of the inverse.
Workspace
DoK 4 — Rungs 7–8 Students can compare and defend multiple strategies, model and analyze situations involving function operations, composition, rates of change, symmetry, or inverses, and evaluate whether conclusions are reasonable in context.
1.

A student says, “If a graph looks symmetric, then it must be even or odd.”

  1. Is this always true?
  2. Use the figure or another example as a counterexample.
  3. Explain the difference between visual symmetry and the algebraic definitions of even and odd functions.
Workspace
2.

Create a function that is symmetric about the \(y\)-axis, has a maximum at \((0,5)\), and is not quadratic.

Write the equation, sketch or describe the graph, and justify why it satisfies all conditions.

Workspace
3.

A school store sells shirts and hats. Let \(S(x)\) be shirt revenue and \(H(x)\) be hat revenue after \(x\) days.

  1. Create reasonable functions \(S(x)\) and \(H(x)\).
  2. Write and interpret \((S+H)(x)\).
  3. Write and interpret \((S-H)(x)\).
  4. Explain whether \(\frac{S(x)}{H(x)}\) has a meaningful interpretation.
Workspace
4.

Use a graph to explain how secant slopes approach a tangent slope as \(h\) gets closer to 0.

  1. Describe what changes in the picture.
  2. Explain what stays fixed.
  3. Connect your explanation to the difference quotient.
Workspace
5.

Design a graph that has an inverse relation but not an inverse function.

  1. Sketch or describe the graph.
  2. Explain why the inverse relation exists.
  3. Explain why the inverse is not a function.
Workspace
6.

Create two functions \(f\) and \(g\) so that:

  1. \((f+g)(x)\) is linear.
  2. \((fg)(x)\) is quadratic.
  3. \(\frac{f}{g}\) has an excluded value.

Explain why your choices satisfy all three conditions.

Workspace
7.

Design a real-world situation involving two steps where the order matters.

  1. Define both functions.
  2. Write both compositions.
  3. Explain why one order matches the situation and the other does not.
Workspace
8.

Design two functions that are inverses of each other.

  1. Write both functions.
  2. Verify by composition.
  3. Explain how the graphs would relate.
