Unit B Practice by Depth of Knowledge

DoK 1 — Rungs 1–2 Students can identify exponent, radical, rational exponent, function notation, and linear-function vocabulary and recognize basic equivalent forms and features.
1.

Match each expression to the property being used.

  1. \(x^4\cdot x^6=x^{10}\)

  2. \(\dfrac{x^9}{x^2}=x^7\)

  3. \((x^5)^3=x^{15}\)

  4. \(x^{-4}=\dfrac{1}{x^4}\)

Properties:

  1. power rule

  2. product rule

  3. negative exponent rule

  4. quotient rule

Workspace
2.

Match each expression to an equivalent radical form.

  1. \(x^{3/2}\)

  2. \(x^{2/5}\)

  3. \(x^{4/3}\)

  4. \(x^{1/6}\)

Choices:

  1. \(\sqrt[5]{x^2}\)

  2. \(\sqrt{x^3}\)

  3. \(\sqrt[6]{x}\)

  4. \(\sqrt[3]{x^4}\)

Workspace
3.

Evaluate each expression without a calculator.

  1. \(\sqrt{49}\)

  2. \(\sqrt{121}\)

  3. \(\sqrt[3]{64}\)

  4. \(\sqrt[4]{81}\)

Workspace
4.

Determine whether each expression is real.

  1. \(\sqrt{-9}\)

  2. \(\sqrt[3]{-8}\)

  3. \(\sqrt[4]{-16}\)

  4. \(\sqrt[5]{-32}\)

Workspace
5.

Identify the slope and \(y\)-intercept of each line.

  1. \(y=3x+5\)

  2. \(y=-2x+7\)

  3. \(y=\frac12x-4\)

  4. \(y=-\frac34x\)

Workspace
6.

State whether each line has positive slope, negative slope, zero slope, or undefined slope.

  1. A line rises from left to right.

  2. A line falls from left to right.

  3. A horizontal line.

  4. A vertical line.

Workspace
7.

If \(f(x)=2x+7\), evaluate each.

  1. \(f(0)\)

  2. \(f(3)\)

  3. \(f(-4)\)

  4. \(f(10)\)

Workspace
8.

Fill in the blanks.

In function notation, \(f(3)\) means the ______ value when the input is ______.

The expression \(f(x+2)\) means replace every \(x\) in the formula with ______.

Equivalent expressions have the same value for every allowed ______.

DoK 2 — Rungs 3–4 Students can apply exponent properties, convert and evaluate rational exponents and radicals, simplify radical expressions, write linear equations, and evaluate functions.
1.

Simplify each expression. Write answers with positive exponents.

  1. \(\dfrac{(2x^3)^2}{4x}\)

  2. \(\dfrac{(3a^2b)^2}{9ab^{-1}}\)

  3. \(\dfrac{(-2m^{-1}n^3)^3}{4m^2n}\)

Workspace
2.

Evaluate each expression without a calculator. Show whether you take the root first or apply the power first.

  1. \(36^{3/2}\)

  2. \(27^{2/3}\)

  3. \(16^{5/4}\)

  4. \(100^{3/2}\)

Workspace
3.

Simplify by first simplifying each radical, then combining like radicals.

  1. \(\sqrt{20}+\sqrt{45}\)

  2. \(\sqrt{18}+\sqrt{50}\)

  3. \(3\sqrt{12}-\sqrt{75}\)

  4. \(2\sqrt{27}+5\sqrt{48}\)

Workspace
4.

Rationalize the denominator.

  1. \(\dfrac{5}{\sqrt{2}}\)

  2. \(\dfrac{3}{\sqrt{7}}\)

  3. \(\dfrac{2}{3\sqrt{5}}\)

  4. \(\dfrac{6}{\sqrt{12}}\)

Workspace
5.

The table shows a linear relationship.

\(x\)01234
\(y\)2581114

  1. Find the slope.

  2. Identify the \(y\)-intercept.

  3. Write an equation.

    \(y\)610141822

    \(y\)25101726

  1. \(g(3)\)

  2. \(g(8)\)

  3. \(g(0)\)

  4. Explain why \(g(2)\) is undefined.

  2. \((x-4)^2=25\)

  3. \(\sqrt{x+3}=6\)

  4. \(\frac{x+5}{3}=8\)

DoK 3 — Rungs 5–6 Students can justify algebraic steps, analyze errors, compare representations, explain structure, and connect equations, tables, graphs, and verbal descriptions.
1.

A student simplifies:

\[\dfrac{4x^{-3}y^2}{8x^2y^{-4}}\]

as

\[\dfrac{1}{2}x^{-1}y^{-2}.\]

Identify the error and simplify correctly using positive exponents.

Workspace
2.

A student evaluates:

\[(-27)^{2/3}\]

and gets \(-9\).

  1. Explain the likely mistake.

  2. Evaluate the expression correctly.

  3. Explain why the result is positive.

Workspace
3.

Compare the expressions \(\sqrt{18}+\sqrt{8}\) and \(\sqrt{26}\).

  1. Are they equivalent?

  2. Simplify the first expression.

  3. Explain why radicals cannot usually be combined by adding radicands.

Workspace
4.

The expression \(\sqrt{x-2}\) is real for some values of \(x\), but not for others.

  1. Find the values of \(x\) that make the expression real.

  2. Explain why \(x=1\) does not work.

  3. Explain why cube-root expressions do not have the same restriction.

Workspace
5.

A student says:

“The slope of the line through \((2,5)\) and \((6,13)\) is \(\frac{2}{4}\) because \(6-2=4\) and \(13-5=8\), so I put the smaller number on top.”

Identify the error and find the correct slope.

Workspace
6.

A student says:

“A vertical line has slope \(0\).”

Explain the mistake. Compare a vertical line and a horizontal line.

Workspace
7.

The same function is represented three ways.

Equation: \(f(x)=2x-3\)

Table:

\(x\)\(0\)\(1\)\(2\)\(3\)
\(f(x)\)\(-3\)\(-1\)\(1\)\(3\)

Description: “Double the input, then subtract \(3\).”

Explain how the equation, table, and description all show the same relationship.

Workspace
8.

A student is solving:

\[(x-2)^2+5(x-2)+6=0.\]

Student A expands first. Student B lets \(u=x-2\).

  1. Solve using substitution.

  2. Solve by expanding.

  3. Explain which method is more efficient and why.

DoK 4 — Rungs 7–8 Students can synthesize exponent, radical, linear, and function-readiness reasoning to solve unfamiliar problems, defend strategies, and transfer between representations.
1.

The expression

\[\dfrac{2^x\cdot 8^{x+1}}{4^{2x-1}}\]

can be rewritten as a single power of \(2\).

  1. Rewrite every factor with base \(2\).

  2. Simplify to a single power of \(2\).

  3. Explain why using the same base makes the expression easier to simplify.

Workspace
2.

A student is evaluating:

\[\left(\dfrac{81}{16}\right)^{3/4}.\]

  1. Rewrite the expression using radical notation.

  2. Evaluate it exactly.

  3. Explain whether it is more efficient to take the fourth root first or cube first.

  4. Defend your strategy.

Workspace
3.

Simplify and rationalize the denominator.

\[\frac{\sqrt{18}+\sqrt{50}}{\sqrt{2}}\]

  1. Simplify the numerator first.

  2. Simplify the quotient.

  3. Explain why simplifying before rationalizing can make the work easier.

Workspace
4.

A rectangle has side lengths \(2\sqrt{3}+1\) and \(3\sqrt{3}-2\).

  1. Write an exact expression for the area.

  2. Simplify the area completely.

  3. Write an exact expression for the perimeter.

  4. Simplify the perimeter completely.

  5. Explain why exact radical form is more useful than decimal approximations here.

Workspace
5.

A student works at a summer job. After \(3\) hours, the student has earned \(\$45\). After \(8\) hours, the student has earned \(\$120\).

  1. Write a linear equation relating earnings \(E\) to hours \(h\).

  2. Interpret the slope.

  3. Interpret the \(E\)-intercept.

  4. Use the equation to predict earnings after \(12\) hours.

  5. Explain whether the model makes sense for all possible values of \(h\).

Workspace
6.

A phone plan charges a fixed monthly fee plus a cost per gigabyte of data. The total cost is \(\$38\) for \(4\) GB and \(\$62\) for \(10\) GB.

  1. Write a linear model for total cost \(C\) in terms of data \(g\).

  2. Interpret the slope in context.

  3. Interpret the \(C\)-intercept in context.

  4. Predict the cost for \(15\) GB.

  5. Explain why negative data values do not make sense in this context.

Workspace
7.

A table, description, and equation are shown.

Table:

\(x\)\(0\)\(1\)\(2\)\(3\)\(4\)
\(y\)\(5\)\(8\)\(11\)\(14\)\(17\)

Description: “Start at \(5\), then increase by \(3\) for every \(1\) increase in \(x\).”

Equation: \(y=3x+5\)

  1. Explain how all three representations match.

  2. Identify the slope and \(y\)-intercept.

  3. Predict the value when \(x=10\).

  4. Find the input when \(y=50\).

  5. Explain why this relationship is linear.

Workspace
8.

A classmate is deciding how to simplify and evaluate:

\[\frac{(x+3)^2-9}{x}\]

when \(x=6\).

  1. Evaluate the expression directly.

  2. Simplify the expression first, then evaluate.

  3. Explain why simplifying first is more efficient.

  4. State any value of \(x\) that is not allowed.

  5. Explain how this problem tests algebraic structure.
